# jokeawhenever.com

Static website files for [jokeawhenever.com](https://www.jokeawhenever.com).

## Overview

This repository contains the **generated static files** for the Joke a Whenever website. It is connected to Cloudflare Pages for automatic deployment.

> ⚠️ **Do not edit files directly in this repository!**
> 
> All content is generated by the [JokeForge-Publish](https://github.com/InquiringKata/JokeForge-Publish) tool. Any manual changes will be overwritten on the next generation.

## Deployment

This repository is connected to Cloudflare Pages:
- **Production URL**: https://www.jokeawhenever.com
- **Build command**: None (static files)
- **Output directory**: `/` (root)

Commits to the `main` branch trigger automatic deployment.

## Updating Content

1. Export jokes from JokeForge database
2. Run JokeForge-Publish to generate new static files
3. Copy generated `output/` contents here
4. Commit and push to trigger deployment

```bash
# From JokeForge-Publish directory
python scripts/generate_site.py
cp -r output/* ../jokeawhenever-site/
cd ../jokeawhenever-site
git add -A
git commit -m "Update site content"
git push
```
